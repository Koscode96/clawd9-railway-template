# Tenant Communications Skill

## Purpose
Handle all tenant communications including enquiries, maintenance requests, rent matters, and general queries for lettings and property management.

## Capabilities
- Answer property/tenancy questions
- Triage and log maintenance requests
- Send rent reminders and chase arrears
- Handle reference requests
- Process move-in/move-out communications
- Respond to complaints professionally

### Common Issues - Quick Responses

**Boiler not working:**
```
Sorry to hear about the boiler. Before we send someone out, could you check:
1. Is the thermostat turned up and set to "on"?
2. Is there credit on the gas meter (if prepay)?
3. Have you tried the reset button (usually on the front panel)?
4. Is the pressure gauge between 1-1.5 bar?

If you've tried all that and it's still not working, I'll arrange an engineer visit.
```

**Blocked drain:**
```
For blocked drains, you can often fix it yourself:
1. Try a plunger first
2. Pour boiling water down the drain
3. Use a drain unblocker product from any supermarket

If it's still blocked after trying these, or if it's the main drain/multiple drains are affected, let me know and I'll get someone out.
```

**Smoke alarm beeping:**
```
That beeping usually means the battery is low. The fix:
1. Press and hold the test button to stop the beeping
2. Replace the 9V battery
3. Test it works by pressing the button again

If it's a hard-wired alarm or keeps beeping with a new battery, let me know.
```

## Reference Requests

### Employer Reference Chasing
```
Hi,

We're processing a rental application for [Applicant Name] who has listed your organisation as their employer.

Could you kindly confirm:
1. Their job title
2. Employment start date
3. Annual salary
4. Is their employment permanent/fixed-term/probationary?

This information is required for referencing purposes and will be kept confidential.

If you'd prefer to complete a formal reference form, I can send one over.

Many thanks!
```

### Previous Landlord Reference
```
Hi,

[Applicant Name] has applied to rent a property through us and has given your details as their current/previous landlord.

Could you please confirm:
1. Tenancy dates (start and end)
2. Monthly rent amount
3. Were payments made on time?
4. Was the property left in good condition?
5. Would you rent to them again?

Your response is much appreciated and will be treated confidentially.
```

### Chasing References (No response after 3 days)
```
Hi,

Just following up on my reference request for [Applicant Name] sent on [date].

The applicant is keen to proceed and we're just waiting on this reference to complete their application.

Is there anything I can do to help speed this up?
```

## Compliance Notes

- Right to Rent: Never process tenants without valid immigration status checks
- Data Protection: Only share tenant info with authorised parties
- Section 21/Section 8: Never issue notices - always escalate to property manager
- Deposit disputes: Always recommend TDS/DPS resolution process
- Gas Safety: Annual checks are legal requirement - chase tenants for access
