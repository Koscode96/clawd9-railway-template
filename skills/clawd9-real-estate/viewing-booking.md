# Viewing Booking Skill

## Purpose
Schedule property viewings, send confirmations, manage reminders, and collect post-viewing feedback.

## Capabilities
- Check calendar availability
- Book viewing appointments
- Send confirmation messages
- Send reminder messages (24hr and 2hr before)
- Collect and route feedback
- Suggest alternative properties if viewing cancelled

## Feedback Handling

**Positive Feedback:**
```
That's great to hear! [Property Address] is getting a lot of interest.

Would you like to:
a) Book a second viewing (perhaps at a different time of day)?
b) Discuss making an offer?
c) See similar properties in the area?

Let me know and I'll arrange it.
```

**Negative Feedback:**
```
Thanks for being honest - it's really helpful to know what works and what doesn't.

Based on what you've said, I think you might prefer [alternative property suggestion] which has [feature they wanted] / doesn't have [feature they disliked].

Would you like me to send you the details?
```

**No Response to Feedback Request (Follow up after 24 hours):**
```
Hi [Name], just checking in after your viewing at [Property Address] yesterday.

Would love to hear your thoughts when you have a moment - even a quick "loved it" or "not for me" helps!

Any questions I can answer?
```

## Calendar Integration

When checking availability, consider:
- Agent working hours (typically 9am-6pm, some Saturdays)
- Travel time between properties (minimum 30 mins between viewings)
- Property access restrictions (tenant notice periods, key collection)
- Block viewings where possible (multiple viewers, same time slot)

## Viewing Types

**Accompanied Viewing (Default):**
- Agent meets client at property
- Full tour and Q&A
- 20-30 minutes typical duration

**Block Viewing:**
- Multiple parties, same time slot
- Creates urgency
- Used for high-demand properties

**Virtual Viewing:**
- Video call walkthrough
- For out-of-area buyers
- Pre-qualifies before in-person visit

**Second Viewing:**
- Bring partner/family
- More detailed inspection
- Often leads to offers

## Compliance Notes

- Confirm ID for all viewings (anti-money laundering)
- Tenant-occupied properties: ensure proper notice given
- Vacant properties: security protocols for key handling
- Never leave viewers unattended in occupied properties
